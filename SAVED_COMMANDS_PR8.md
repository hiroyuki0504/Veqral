# PR #8: 定型コマンド保存/再送

## 目的

よく使う Command 下書きを保存し、タップで composer に戻せるようにした。保存先は通常のアプリ状態に加え、iCloud Documents が利用可能な端末では `Veqral/saved-command-drafts.json` に best-effort で同期キャッシュする。iCloud が未設定・未許可の環境ではローカルキャッシュへ安全にフォールバックする。

## 実装

- `SavedCommandDraft` を追加し、command 本文・表示タイトル・実行方式・作成/更新日時を保持。
- Command composer に「定型コマンド」バーを追加。
  - ブックマーク追加ボタンで現在の指令を保存。
  - 保存済み chip をタップすると指令欄へ再投入し、保存時の実行方式も復元。
  - menu から挿入/削除。
- 既存 `CommandCenterSnapshot` にも保存済み定型を含め、iCloud キャッシュが無い環境でも復元できるようにした。
- 保存件数は最新 24 件まで。本文+実行方式が同じ場合は更新扱い。

## 受け入れ

- 指令欄に文を入力して保存すると、定型コマンドバーに表示される。
- 保存済み chip をタップすると composer に同じ文が戻る。
- iCloud が使える端末では同期キャッシュへ書き込み、使えない端末ではローカル状態だけで動作する。
